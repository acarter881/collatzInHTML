The Collatz conjecture

Despite recent progress on the notorious Collatz conjecture, we still don’t know whether a number can escape its infinite loop.
Patrick Honner, 2020 - The Simple Math Problem We Still Can’t Solve

The Collatz conjecture (a.k.a. the 3n + 1 problem) is, as of Saturday, October 24th, 2020, an unsolved math problem.

The Collatz conjecture takes any positive integer n and does one of two actions to it:

If n is even, divides n by 2.
If n is odd, multiplies n by 3 and adds 1.

Repeating this process for the return value of each previous iteration eventually results in a final output of 1.
Laypeople may think that following the KISS method would be enough to solve this math problem, but that is NOT the case.

For example, let's assume n is equal to 12:

Since 12 is an even number, we'll divide n by 2, to get 6.
Since 6 is also an even number, we'll divide 6 by 2, to get 3.
Since 3 is an odd number, we'll multiply 3 times 3 and add 1, to get 10.
Since 10 is an even number, we'll divide 10 by 2, to get 5.
Since 5 is an odd number, we'll multiply 5 times 3 and add 1, to get 16.
Since 16 is an even number, we'll divide 16 by 2, to get 8.
Since 8 is an even number, we'll divide 8 by 2, to get 4.
Since 4 is an even number, we'll divide 4 by 2, to get 2.
Since 2 is an even number, we'll divide 2 by 2, to get 1.

The Collatz conjecture takes 9 iterations to go from 12 to 1.

Using Python to figure out the iterations it takes for a positive integer n to get to 1.

The code at the bottom of the page starts off by defining a function called collatz which takes a single input (an integer n).
When the program is run, the user is prompted to input a positive integer; this integer is stored in the user_number variable.
After this, the program runs until the return value of collatz(user_number) is equal to 1, printing n and the return value for each iteration from n to 1, inclusive.
The final output of the program is a line of text that tells you the number of iterations required to convert n to 1.

If the user inputs 12 for the value of n, the program's output will be as follows:
